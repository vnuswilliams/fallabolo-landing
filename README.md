# MatchRH - Recrutement Intelligent au Mérite

MatchRH est une plateforme de recrutement innovante conçue pour le marché camerounais, visant à éliminer le bruit et les biais du tri manuel de CV traditionnel au profit d'un **matching algorithmique transparent et structuré**.

## 🚀 La Vision

Le constat est simple : un recruteur passe en moyenne 6 secondes par CV, tandis qu'un candidat passe des heures à peaufiner un PDF. MatchRH inverse cette dynamique en remplaçant le CV par un **profil structuré** et un **score de compatibilité instantané**.

## ✨ Fonctionnalités Clés

### 🎯 Pour les Recruteurs (Entreprises & PME)
- **Candidatures Qualifiées uniquement** : Fini le tri de centaines de PDFs inadaptés.
- **Critères Bloquants** : Définissez des exigences non-négociables (permis, localisation, expérience min.) qui filtrent automatiquement les profils.
- **Classement Automatique** : Les candidats arrivent déjà triés du plus compatible au moins compatible.
- **Points Bonus** : Valorisez les expertises rares ou certifications via un système de bonus pondéré.
- **Gratuité Totale** : Pas de barrière à l'entrée pour publier des offres illimitées.

### 💼 Pour les Candidats (Talents)
- **Recrutement sans CV obligatoire** : Créez un profil basé sur vos compétences réelles, pas sur vos talents de designer.
- **Transparence Totale** : Visualisez votre score de compatibilité **avant** même de postuler.
- **Auto-sélection** : Ne perdez plus de temps sur des offres qui ne vous correspondent pas.
- **Évaluation au Mérite** : L'algorithme ignore le nom, l'âge ou la photo pour se concentrer sur les compétences et l'expérience.

## 🧠 L'Algorithme de Matching

Le scoring de MatchRH est **déterministe et transparent** (pas de "boîte noire"). Il repose sur 6 dimensions clés :

1. **Compétences (50%)** : Évaluation technique notée sur 5.
2. **Expérience (20%)** : Parcours vérifiable.
3. **Formation (10%)** : Cursus académique.
4. **Langues (10%)** : Aptitudes linguistiques.
5. **Disponibilité (5%)** : Immédiateté ou préavis.
6. **Localisation (5%)** : Proximité géographique.

## 🛠️ Stack Technique

- **Framework** : Laravel 11
- **Frontend** : Livewire, Alpine.js, Tailwind CSS
- **UI Components** : Flux UI
- **Documentation** : Markdown-based legal pages

## 🔮 Vision Futur : MCP Integration

MatchRH prévoit l'intégration du protocole **MCP (Model Context Protocol)**. Cela permettra aux utilisateurs d'interagir avec la plateforme directement depuis des assistants IA (Claude, ChatGPT, Cursor) :
- "Trouve-moi les 3 meilleurs développeurs Laravel à Douala."
- "Publie une offre pour un comptable senior avec 5 ans d'expérience."

## 📜 Légal & Données

MatchRH est en conformité avec la **loi camerounaise de 2024 sur la protection des données personnelles**. Les données sont sécurisées, tracées et le profil est supprimable à tout moment par l'utilisateur.
